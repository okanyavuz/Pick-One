# Runner Math Challenge - Running Mathematical Adventure

## Game Description

Runner Math Challenge is a mathematical adventure game where players automatically run upon game start. As the character runs, they navigate through red and green wall-like structures. The character's speed and score are displayed on top, representing their progress. The red and green walls display simple arithmetic operations, contributing to the character's number displayed. However, players need to ensure that the number on the character doesn't reach zero. The game progresses through various levels, and the primary objective is to achieve a high score.

## Features

- **Automatic Running**: The character automatically runs from the beginning.
- **Mathematical Walls**: Red and green walls feature simple arithmetic operations.
- **Number Management**: Players must manage the number on the character to avoid reaching zero.
- **Level-based Progression**: The game advances through different levels, offering new challenges.
- **High Score Challenge**: The primary goal of the game is to achieve a high score by effectively managing numbers and progressing through levels.
